ASSEMBLER

******************************
Tools Used: Assembler, CPUEmulator

hackassembler.py is a Python script that will translate an *.asm assembly language file into a *.hack machine code file.

To run hackassembler.py from a Windows command line, use the syntax:
<python.exe pathname> <hackassembler.py pathname> <*.asm pathname>

RUN.bat is a batch file that will generate a *.hack file in each of the four project06 folders (add, max, pong, rect).
RUN.bat can be started from the command line by typing its pathname.
