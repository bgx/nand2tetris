MACHINE LANGUAGE

Get a taste of low-level programming in machine language, and get acquainted with the Hack computer platform.

******************************

--Write and test an Abc.asm program--
1. Write program Abc.asm in assembly
2. Use the supplied assembler to translate Abc.asm to Abc.hack, which contains binary machine instructions
3. Use the supplied CPU emulator to test the resulting Abc.hack code (either load the Abc.tst file and run, or load the Abc.hack file and run)
